# FlySightWebTool
Web-based FlySight ([flysight.ca](https://flysight.ca/)) track viewer and device configuration editor (WIP).

**Dependencies:**
* Net8 or later
* Environment variable 'GoogleMapsApiKey' containing your Google Maps API key
